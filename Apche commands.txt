Apache Commands

1)To start Apache web server.
sudo systemctl start apache2

2)To stop Apache webserver.
sudo systemctl stop apache2

3)To restart Apache web server.
sudo systemctl restart apache2

4)To reload Apache without dropping connections.
sudo systemctl reload apache2

5)To disable Apache.
sudo systemctl disable apache2

6)To enable Apache.
sudo systemctl enable apache2
